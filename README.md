# Fantasy Map Generator (奇幻架空地图生成器)

这是一个基于纯前端技术 (HTML5 Canvas + JavaScript) 开发的程序化架空世界地图生成器。它可以生成极具真实感、符合自然地理法则的大陆板块、气候带与水文环境。

## ✨ 核心特性

*   **🌍 无缝 3D 圆柱投影**：完美适配 2:1 宽高比（1600x800），解决传统 2D 噪声在地图左右两端无法缝合以及两极极度拉伸的数学问题。
*   **🏔️ 真实的板块构造**：基于改进的 Voronoi 算法划分地壳板块，模拟真实的板块碰撞与造山运动，并在大陆边缘自动形成**火山岛弧与岛链**。
*   **🌊 全局动态海平面**：精准把控 3:7 的海陆比例，动态计算海平面高度。支持从极高山到深海的多级等高线地形渲染。
*   **❄️ 纬度与气候带**：结合地形海拔与纬度计算温度与湿度，两极区域自动生成逼真的**极地冰盖**过渡带。
*   **💧 高级水文系统 (Hydrology)**：
    *   基于重力与降水概率计算源头，模拟水滴汇聚过程。
    *   加入自然微扰的寻路算法，生成蜿蜒曲折的长河。
    *   内陆断流处基于柏林噪声生成自然侵蚀边缘的**淡水湖泊**。
    *   **矢量抗锯齿渲染**：彻底消除像素感，通过动态宽度平滑绘制源头至入海口的渐变水系。
*   **🗺️ 多种可视化模式**：支持“地形混合”、“政治版图”、“纯地形”、“纯等高线”四种视图。支持经纬网格叠加。
*   **📸 高清导出**：自带完整的标准测绘地形图例，支持在移动端长按导出带图例的高清成品大图。

## 🚀 运行方式

无需任何构建工具或依赖，直接在浏览器中打开 `index.html` 即可运行。

如需在手机端（如 Termux）本地启动 Web 服务预览：
```bash
python3 -m http.server 8080
```
然后在浏览器访问 `http://localhost:8080` 即可。

## 🛠️ 技术栈

*   原生 JavaScript (ES6+)
*   HTML5 `<canvas>` API
*   [simplex-noise.js](https://github.com/jwagner/simplex-noise.js) (柏林噪声生成)
